Truc: fes servir la cerca ràpida de dalt (o Ctrl/⌘+F) per saltar directament al que necessites.

🔢 # Aritmètica

  • Operacions bàsiques: suma, resta, multiplicació, divisió; prioritat (parèntesis → ×/÷ → +/−).
  • Negatius: regles de signes (−×− = +, +×− = −, etc.).
  • Divisions exactes: factors i múltiples; comprova si el divisor “cap” al dividend.
  • Trucs ràpids: propietats associativa, commutativa i distributiva.

🍰 # Fraccions

  • Parts: numerador/denominador; fracció equivalent; simplificar amb mcd.
  • Aritmètica:
    • a/b + c/d = (ad+bc)/bd
    • a/b − c/d = (ad−bc)/bd
    • a/b × c/d = ac/bd
    • a/b ÷ c/d = a d / b c (multiplica pel recíproc)
  • De decimal a fracció: 0.25 → 25/100 → 1/4.
  • Fracció de dibuix: parts ombrejades / parts totals (i simplificar).
  • Quan exigeix forma més simple: el numerador i el denominador han de ser coprimers.

％ # Percentatges

  • Definició: p% = p/100.
  • Tres tipus típics:
    1. “p% de n” → n·p/100
    2. “k és el p% de ?” → ? = k·100/p
    3. Descompte p% sobre n → n·(1 − p/100)
  • Canvis successius no es sumen: 20% + 20% ≠ 40% (usa factors multiplicatius).

🟦 # Geometria (àrea, perímetre, volum)

  • Unitats: longitud u, àrea u², volum u³. Escriu la unitat si el programa ho demana.
  • Formules clau:
    • Rectangle: A = b·h, P = 2(b+h)
    • Triangle: A = (b·h)/2, P = a+b+c
    • Cercle: A = πr², P = 2πr = πd
    • Polígon regular: P = n·c; A = (P·apotema)/2
    • Prisma rectangular (cuboid): V = a·b·h
    • Cilindre: V = πr²h
  • Exacte amb π: deixa el resultat com k·π si l’opció diu “π exacte”.
  • Arrodoniments: respecta els decimals que demana la configuració.

📊 # Estadística bàsica

Donada una llista de valors:

  • Mitjana: suma/quantitat.
  • Mediana: valor central (o mitjana dels dos centrals).
  • Moda: valor que més es repeteix.
  • Rang: màxim − mínim.
  • Desviació mitjana: mitjana de |xi − mitjana|.

Gràfics: identifica la barra/sector més gran.

🔁 # Unitats i conversions

  • Sistema mètric:
    • Longitud: km ↔ m ↔ cm ↔ mm (×10, ×100, ×1000…)
    • Massa: kg ↔ g ↔ mg
    • Volum: L ↔ mL
    • Àrea: factors al quadrat (1 m² = 10 000 cm²)
    • Temps: h ↔ min ↔ s (60 i 3600)
  • Mètode: valor · factor (o / factor si és el camí invers). Vigila amb u², u³.

🧮 # Equacions

1r grau ax + b = 0

Passa termes i divideix: x = −b/a.

2n grau ax² + bx + c = 0

  • Factorització si es pot, o fórmula general: x = (−b ± √(b² − 4ac)) / (2a)
  • Incompletes: ax² + c = 0 → x = ±√(−c/a); ax² + bx = 0 → x(ax + b)=0 → x=0 o x=−b/a.
  • Discriminant Δ=b²−4ac (2, 1 o 0 arrels reals).

Sistemes 2×2

Substitució o reducció; comprova casos sense solució o infinites (rectes paral·leles/ coincidents).

Equacions amb fraccions

Troba mcd/mcm i multiplica per l’mcm per eliminar denominadors, després resol.

Parèntesis

Distributiva, agrupa termes semblants, i resol com una de 1r grau.

📈 # Funcions (estudi bàsic)

Identificació:

  • Lineal mx+n; Quadràtica ax²+bx+c; Polinòmica grau n; Racional P(x)/Q(x);
  • Radical a√(x+b)+c; Exponencial a·b^x + c; Logarítmica a·log_b(x)+c.

Domini:

  • Lineal/quadràtica/polinòmica: ℝ
  • Racional: exclou zeros del denominador
  • Radical: interior de l’arrel ≥ 0 (si és √ parell)
  • Logarítmica: x > 0

Punts de tall:

  • Amb y: posa x=0; amb x: resol f(x)=0.

Simetria i extrems (quadràtiques):

  • Simetria respecte x = −b/2a.
  • Vèrtex x_v = −b/2a; si a>0 mínim, si a<0 màxim.

Monotonia:

  • Lineal creix si m>0, decreix si m<0; quadràtica canvia en el vèrtex.

🧾 # Formats de resposta (molt important en aquesta app)

  • Nombres i decimals: millor amb punt (p. ex. 3.5).
  • Fraccions: a/b (simplificades si es demana “forma més simple”).
  • π exacte: escriu kπ (p. ex. 12π).
  • Unitats (geometria): si l’opció “exigir unitats” està activa, afegeix cm, cm², cm³…
  • Equacions de 1r/2n grau: pots escriure x=3 o 3; per a dues arrels, separa amb coma x1,x2.
  • Sistemes 2×2: (x, y) o x,y (p. ex. (2, -1)).
  • Conversions: escriu només el número (la unitat ja surt a l’enunciat).

⚠️ # Errors típics a evitar

  • Oblidar unitats o el quadradet/cubet en àrees/volums.
  • No simplificar fraccions quan cal.
  • Confondre d i r en cercles (P=πd, A=πr²).
  • Sumar/descomptar percentatges en comptes d’usar factors multiplicatius.
  • En 2n grau, oblidar comprovar el discriminant o canviar signes.
  • En sistemes, no revisar si hi ha paral·lelisme (cap solució) o coincidència (infinites).
